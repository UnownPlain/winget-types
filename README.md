# WinGet Types

[![CI Status](https://github.com/russellbanks/winget-types/workflows/CI/badge.svg)](https://github.com/russellbanks/winget-types/actions)

## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
winget-types = "0.1"
```
